🌆 Generative Terminal World — Infinite Cyberpunk City in Your Terminal

Just Shipped | Python, Rich, Procedural Generation

Code: GitHub

A goofy, genuinely fun side project: walk around an endlessly-generating neon city rendered entirely in your terminal. Streets, alleys, plazas, markets, corporate sectors, slums, rooftops, and clubs — with random street events as you explore.

The trick: every coordinate is generated on the fly with a seeded RNG (md5 hash of its x:y position), so the world is infinite but stable — walk away and come back, the same block is exactly where you left it, with nothing pre-stored.

WASD/arrow-key movement, built on rich for the terminal rendering. Actively growing — this is the "I built this because it was fun" shelf, and it's staying open.

🤖 Hermes Agent — Local LLM Bitcoin Content Swarm

Python, Ollama (local inference), Tweepy, Cron

A 4-agent "swarm" that researches, writes, and posts Bitcoin education content as @_bitcoin_basics — running entirely on a local hermes3 model via Ollama, zero cloud LLM cost. Pipeline: pick topic → research → write (with a custom voice/style rules engine) → post to X → email recap. Cron-scheduled daily, plus a manual approve-before-post mode.

Also builds automation with n8n (self-hosted, workflow-based monitoring/alerting) and integrates MCP tooling into daily dev workflow.
